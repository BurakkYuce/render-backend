=== RENTALY BACKEND API ENDPOINTS ===
Base URL: http://localhost:3001/api

=== CARS APIS ===
GET /cars/filtered - Get filtered cars with pagination
Example: GET /cars/filtered?vehicleType=SUV&page=1&limit=12

GET /cars/filter-options - Get available filter options  
Example: GET /cars/filter-options

POST /cars/:id/like - Toggle car like/favorite
Example: POST /cars/123/like

GET /cars/search - Search cars by query
Example: GET /cars/search?q=toyota&limit=10

GET /cars/:id - Get single car details
Example: GET /cars/123

POST /bookings - Create new booking
Example: POST /bookings
Body: {"carId": "123", "startDate": "2025-01-01", "endDate": "2025-01-07"}

GET /locations - Get available pickup locations
Example: GET /locations

GET /cars/:id/availability - Check car availability
Example: GET /cars/123/availability?startDate=2025-01-01&endDate=2025-01-07

=== NEWS APIS ===
GET /news - Get all published news articles
Example: GET /news?page=1&limit=10

GET /news/:id - Get single news article
Example: GET /news/123

GET /news/recent - Get recent news posts
Example: GET /news/recent?limit=5

GET /news/tags/popular - Get popular news tags
Example: GET /news/tags/popular

GET /testimonials - Get customer testimonials
Example: GET /testimonials

GET /news/:id/comments - Get news article comments
Example: GET /news/123/comments

POST /news/:id/comments - Post new comment on news
Example: POST /news/123/comments
Body: {"name": "John", "email": "john@email.com", "comment": "Great article!"}

GET /company/about - Get company information
Example: GET /company/about

POST /social/share/:platform/:itemId - Track social media shares
Example: POST /social/share/facebook/123

=== ADMIN CAR MANAGEMENT ===
GET /admin/cars - Get cars for admin management
Example: GET /admin/cars?page=1&limit=10
Auth: Required

GET /admin/cars/:id - Get single car for admin editing
Example: GET /admin/cars/123
Auth: Required

POST /admin/cars - Create new car
Example: POST /admin/cars
Body: {"name": "Toyota Camry", "basePrice": {"USD": 50}}
Auth: Required

PUT /admin/cars/:id - Update existing car
Example: PUT /admin/cars/123
Body: {"name": "Updated Toyota Camry", "basePrice": {"USD": 55}}
Auth: Required

DELETE /admin/cars/:id - Delete car
Example: DELETE /admin/cars/123
Auth: Required

GET /admin/cars/:id/pricing - Get scheduled pricing
Example: GET /admin/cars/123/pricing
Auth: Required

POST /admin/cars/:id/pricing - Add scheduled pricing
Example: POST /admin/cars/123/pricing
Body: {"startDate": "2025-06-01", "endDate": "2025-08-31", "price": {"USD": 75}}
Auth: Required

DELETE /admin/cars/:carId/pricing/:pricingId - Delete scheduled pricing
Example: DELETE /admin/cars/123/pricing/456
Auth: Required

=== AUTHENTICATION NOTES ===
- "Auth: Required" endpoints need admin authentication
- Use JWT token in Authorization header: "Bearer YOUR_TOKEN"
- Get token from login endpoint (check existing auth routes)

=== COMMON PARAMETERS ===
- page: Page number for pagination (default: 1)
- limit: Items per page (default: 10-12)
- q: Search query string
- status: Filter by status (active, inactive, draft, published)
- startDate/endDate: Date range filters (format: YYYY-MM-DD)
